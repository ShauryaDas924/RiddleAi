# Project Plan

Riddle AI is a student learning app where users practice riddles, submit answers, receive feedback, and track progress.

The app should stay simple at first:

1. Show a riddle.
2. Let the student answer.
3. Check the answer.
4. Save basic progress later.
5. Avoid showing the same riddle too often later.

Architecture:

React frontend -> FastAPI backend -> Supabase database/auth -> AI provider
