# Student Build Order

1. Understand the file tree.
2. Read the guided TODO comments in the first five files.
3. Fill frontend static pages.
4. Add the backend health route.
5. Connect the frontend to the backend.
6. Add Supabase tables.
7. Add riddle generation.
8. Add riddle history.
9. Add preferences.
10. Add auth later.
11. Add tests after each feature becomes stable.

## Best First Files

- `frontend/my-react-app/src/pages/PlayPage.jsx`
- `frontend/my-react-app/src/components/riddles/RiddleCard.jsx`
- `frontend/my-react-app/src/hooks/useRiddle.js`
- `backend/app/main.py`
- `backend/app/routes/health_routes.py`

## Avoid

Do not try to fill every file in one sitting. Build one small working feature at a time.
